# Toolbox

Personal toolbox for userscripts, browser automation, snippets, and small utilities.

This repository is for small tools that are useful enough to keep under version
control, but not large enough to require their own repositories yet.

## Layout

- `userscripts/`: Browser userscripts for tools such as Tampermonkey.
- `browser/`: Browser-side utilities and automation helpers.
- `snippets/`: Small reusable code fragments.
- `one-off/`: Short-lived scripts that may be archived later.
